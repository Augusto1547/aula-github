# Aula de Git e GitHub

## Comandos

- `git --version:` Retorna a versão do Git instalado.
- `git clone URL:` Cria uma cópia local de um repositório remoto.
- `git add ` adicionar as modificações que eu fiz nesse commit que vou dar em seguida
- `git add . :` Adiciona os arquivos /modificaç~eos realizados no próximo commit.
- `git commit -m`
- `git log` Tras todos os commits com todas as modificações
- `git log --oneline` Faz o mesmo só que traz o resultado mais enchuto
- `git log` --oneline -n 2 (retorna os ultimos dois commits apenas)
- git push origin main: Envia seus commits do repostitório local para o remoto (branch main).
-git pull origin main
